##观点分享

1. 开源与非开源之间没有万仞的差距。

    开源，是一种选择，它不应该有过多的道德荣誉；闭源，是一种选择，它不
   应该接受过多的道德鄙视。开源的，提供给人们的价值不一定高；闭源的，也
   可以带给人们超乎想象的价值。

2. Linux与开源并不是等价关系。

    之前，我所认为的开源项目，指的就是Linux操作系统（或Linux内核），故
   在用词上忽视了其他著名的开源项目。Linux是最经典的、最成功的一个开源
   项目，但它并不是开源世界的全部，虽然很多认识到的事情都是构建在Linux
   基础之上。

    作为一个普通的Linux桌面用户，往往会忽略企业级的开源社区动态。云计算、
    大数据……也充满了开源，可以这么说：开源多样，这个世界远远比我们现在
    所认识到更加精彩！

3. 开源社区或开源项目，若要活得长久，必须至少具备下面其一个要素：

+ 兴趣浓厚
+ 技术领先
+ 商业空间巨大

    兴趣，从我个人看来，认为它是最重要的一个要素。如果没有兴趣，人们在
    其中得不到快乐、找不到可以玩得爽的高潮“G点”，无法“取悦”社区成员，便
    无法获得长久的社区成长。人们不会强迫自己去接受一个没有趣味的怪物。

    技术领先。如果没有雄厚的技术实力，其他的比较优势也少的可怜，那么，
    一个项目是无法真正获得应有的地位的。

    商业空间。我们会看到，很多项目的主要维护者有一份“Day Job”，开源项目
    仅是作为他的业余爱好。如果他是专职来做开源项目，要么他是为公司做公
    司会用到的东西，要么他是为一个拥有巨大商业空间的开源项目来做事情。
